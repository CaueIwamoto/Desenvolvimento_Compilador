# Desenvolvimento de um Compilador

## Sobre
- Projeto da disciplina de Compiladores - FEI 2026. O compilador deve fazer a conversão de um programa desenvolvido em uma linguagem de programação esotérica. A verificação da corretude do programa será realizada compilando o arquivo gerado pelo compilador desenvolvido.

Obs: a gramática não pode conter recursividade à esquerda.
